# Jellyfin 媒体库健康检查 — 检查器配置规范 v1

> 基于用户脚本 `check_thumb.py` 提取的检查规则，去除了自动删除/重命名操作。

## 核心概念

### 目录层级定义

```
base_dir = "/media/movies"                     # 配置的库根路径
         │
         └── action/                           # level 0 → 演员目录
               │
               ├── movie-A/                    # level 1 → 电影目录（检查目标）
               │     ├── movie.nfo
               │     ├── folder.jpg
               │     ├── landscape.jpg
               │     ├── backdrop0.jpg
               │     ├── movie-A.zh.srt
               │     └── extrafanart/          # ← 提示可清理
               │
               └── movie-B/                    # level 1 → 电影目录（检查目标）
                     ├── movie.nfo
                     ├── movie-B.srt            # ← 缺 .zh 前缀
                     └── thumb.jpg              # ← 提示可清理
```

**check_target_level: 1** — 只检查 base_dir 下第 1 级子目录。

### 可扩展设计

不同用户可能有不同目录结构：

| 用户场景 | 层级 | 检查目标 |
|---------|------|---------|
| Jellyfin 标准 | level 1 | 电影目录 |
| 平铺结构 | level 0 | 所有子目录 |
| 电视剧 | level 2 | Season 目录 |

通过修改 `check_target_level` 适配。

## 检查器配置

### 配置格式（JSON）

```json
{
  "name": "jellyfin-movie-structure",
  "version": "1.0",
  "description": "Jellyfin 电影库目录结构健康检查",
  "check_target_level": 1,
  "rules": {
    "required_files": [
      {
        "files": ["folder.jpg", "landscape.jpg"],
        "severity": "warning",
        "message_zh": "缺少封面图片: {missing}",
        "message_en": "Missing cover images: {missing}"
      }
    ],
    "pattern_checks": [
      {
        "pattern": "*backdrop*.jpg",
        "severity": "warning",
        "message_zh": "缺少背景图片(backdrop*.jpg)",
        "message_en": "Missing backdrop image"
      }
    ],
    "cleanup_hints": [
      {
        "path": "extrafanart",
        "type": "directory",
        "severity": "info",
        "message_zh": "存在 extrafanart 文件夹(Jellyfin 会自动生成, 可清理)",
        "message_en": "extrafanart folder exists (auto-generated by Jellyfin, safe to remove)"
      },
      {
        "path": "thumb.jpg",
        "type": "file",
        "severity": "info",
        "message_zh": "存在 thumb.jpg (Jellyfin 会自动生成, 可清理)",
        "message_en": "thumb.jpg exists (auto-generated by Jellyfin, safe to remove)"
      }
    ]
  }
}
```

### 规则说明

| 规则类型 | 逻辑 | 严重度 |
|---------|------|--------|
| `required_files` | 指定文件列表必须**全部**存在，`{missing}` 替换为缺失文件名 | `warning` |
| `pattern_checks` | glob 模式匹配，至少一个文件命中 | `warning` |
| `cleanup_hints` | 指定路径存在则提示 | `info` |

### 严重度定义

| 级别 | 颜色 | 含义 |
|------|------|------|
| `error` | 🔴 | 阻塞性问题（如缺 movie.nfo 无法扫描） |
| `warning` | 🟡 | 建议修复（如缺封面图） |
| `info` | 🔵 | 提示信息（如可清理文件） |

## 扩展示例

### 示例 1：用户有平铺结构（无演员目录）

```json
{
  "name": "flat-movie-structure",
  "check_target_level": 0,
  "rules": {
    "required_files": [
      { "files": ["movie.nfo"], "severity": "error", "message_zh": "缺少 movie.nfo" }
    ]
  }
}
```

### 示例 2：增加必须文件

```json
{
  "rules": {
    "required_files": [
      { "files": ["folder.jpg", "landscape.jpg"], "severity": "warning" },
      { "files": ["movie.nfo"], "severity": "error" }
    ]
  }
}
```

## 实现映射

| 检查规则 | 脚本原始逻辑 | 配置化后 |
|---------|------------|---------|
| folder.jpg + landscape.jpg | `all(file in files for file in required_files)` | `required_files: ["folder.jpg", "landscape.jpg"]` |
| backdrop*.jpg | `any("backdrop" in file.lower() and file.endswith(".jpg"))` | `pattern_checks: ["*backdrop*.jpg"]` |
| extrafanart 文件夹 | `shutil.rmtree(extrafanart_path)` ← 删除 | `cleanup_hints: {path: "extrafanart", type: "directory"}` ← 只提示 |
| thumb.jpg | `os.remove(thumbs_path)` ← 删除 | `cleanup_hints: {path: "thumb.jpg", type: "file"}` ← 只提示 |
